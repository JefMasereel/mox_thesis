This dataset is a devalued sample from the original dataset collected by Nicky Daniels et al. 
This is done to demonstrate the processing pipeline under restricted publication rights. 
Each file contains the same measurements, copied from a nonspecified participant. 

PP02_RS_PRE.txt represents the measurements of subjectID=2, at rest, before treatment.
rawdata_attachment.txt contains a matching sample of placebo & attachment scores.
